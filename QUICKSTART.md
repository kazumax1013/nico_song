# 🚀 クイックスタート：5分で公開

## 最も簡単な方法（推奨）

### Netlifyで公開（ドラッグ&ドロップ）

1. **Netlifyにアクセス**
   ```
   https://www.netlify.com/
   ```

2. **アカウント作成**
   - 「Sign up」をクリック
   - GitHubアカウントでログイン（推奨）

3. **デプロイ**
   - ダッシュボードで「Add new site」→「Deploy manually」
   - **このフォルダ全体をドラッグ&ドロップ**
   - 完了！数秒で公開されます

4. **URL確認**
   - デプロイ後、`https://[ランダムな名前].netlify.app` のURLが表示されます
   - このURLをクリックしてサイトを確認

## その他の方法

### Vercelで公開

1. https://vercel.com/ にアクセス
2. GitHubアカウントでログイン
3. 「Add New Project」→ フォルダをドラッグ&ドロップ
4. 「Deploy」をクリック

### GitHub Pagesで公開

1. GitHubで新しいリポジトリを作成
2. このフォルダの内容をアップロード
3. Settings → Pages → Source を「main」ブランチに設定
4. 数分後に `https://[ユーザー名].github.io/[リポジトリ名]` でアクセス可能

## ⚠️ 注意事項

- 動画ファイル（`images/hero/マイビデオ.mp4`）が大きい場合、読み込みが遅くなる可能性があります
- 初回デプロイ後、すべての画像と動画が正しく表示されるか確認してください
